# Lesser Helix

A very compact, split, ortholinear, hot-swappable keyboard designed for use
with the nice!nano.

![Lesser Helix](README-photo.png)
![Lesser Helix Kicad](README-kicad.png)

## Features

- Hotswap choc and mx sockets
- No TRRS jack
- Battery switch
